## Extension Settings
This extension contributes the following settings:

* `geekLanguage.regexp`: 符合条件代码块的正则匹配查询, 默认情况下为`$L(${language}),t(${language})`
* `geekLanguage.resources`: 转译的资源路径, 支持`*.json, *.js`
